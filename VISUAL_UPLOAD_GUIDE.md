## WHAT IF SOMETHING GOES WRONG?

### Website Not Showing?

```
Problem: After 2 minutes, website still blank
Solution:
1. Hard refresh browser: Ctrl+Shift+R (Windows)
                       Cmd+Shift+R (Mac)
2. Wait another 30 seconds
3. Try in an incognito/private tab (bypasses cache)
```

### Page Looks Broken/Missing Styles?

```
Problem: Page shows text but no colors/styling
Solution:
1. Check that styles.css is in your repo
2. Make sure filename is exactly: styles.css (no spaces)
3. Check spelling: index.html (not Index.html or index.HTML)
4. Delete files and try uploading again
```

### Nothing Shows Up?

```
Problem: Page is completely blank
Solution:
1. Go to your repo settings:
   GitHub repo â†’ Settings â†’ Pages
2. Check "Source" is set to "Deploy from a branch"
3. Check "Branch" is set to "main" or "master"
4. Check "Folder" is set to "/ (root)"
```
